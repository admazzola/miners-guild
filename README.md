## Miners Guild Contract 


#### Getting Started
 
This contains the contract and unit tests for the Miners Guild DAO. 

#### Commands: 
    nvm install 14 (requires node14)
    nvm use 14

    npm install 
    

    npm run compile 
    npm run test
